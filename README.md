# Aria Rahimi Site

This folder is a publish-ready export of the public personal site.

## Publish Flow

1. Export from the private workspace source in site-draft/.
2. Review the files here before publishing.
3. Commit and push only public-safe files to khalkhali/ariarahimi-site.
4. Publish the site from the public repo main branch through GitHub Pages.
5. Keep the custom domain set to ariarahimi.com.

## Public Repo Rules

- Do not copy private planning docs or notes into this repo.
- Do not publish certificates, exports, or internal workspace files here.
- This repo is intentionally public, so source view and forking should be assumed.
