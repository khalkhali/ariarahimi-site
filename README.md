# Aria Rahimi Site

This folder is a publish-ready export of the personal site.

## Publish Flow

1. Create a separate public GitHub repository for the website.
2. Copy the contents of this folder into the root of that repo.
3. Push to main.
4. Enable GitHub Pages from the main branch.
5. In GitHub Pages settings, set the custom domain to riarahimi.com.
6. In Squarespace DNS, replace the default Squarespace web records with GitHub Pages records.
